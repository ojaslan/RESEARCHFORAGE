## 🚀 Overview

Paper Intelligence is an AI-powered research paper analyzer that helps students, researchers, and professionals understand research papers in minutes.

Simply upload a PDF research paper, and the application automatically extracts metadata, summarizes the paper, explains the methodology, identifies strengths and limitations, and generates a structured report using Groq's Llama 3.3 language model.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/Paper-Intelligence.git

cd Paper-Intelligence
```

---

## Create Virtual Environment

### Windows

```bash
python -m venv venv

venv\Scripts\activate
```

### Linux / Mac

```bash
python3 -m venv venv

source venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

# 🔑 Configure API Key

Create a `.env` file
